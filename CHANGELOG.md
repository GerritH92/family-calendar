# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-11-26

### Added
- Initial release of Family Calendar integration
- Weekly calendar view with multi-calendar support
- Add and delete events functionality
- Weather forecast integration
- Custom calendar colors and names
- Responsive design for all devices
- Config flow for easy setup
- HACS support

### Features
- Aggregate events from multiple calendars
- Filter calendars with toggle pills
- Interactive event details view
- Live clock display
- Support for Google Calendar, Local Calendar, and CalDAV
